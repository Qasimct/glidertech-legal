Self-Help First

Before Contacting Support

These steps resolve the majority of common issues across all Gliders Tech applications.

🔄

Update the Application

Ensure you are using the latest version of the application. Most bug fixes and improvements are delivered through updates on Google Play.

🔁

Restart Your Device

A device restart clears temporary memory issues and resolves many intermittent problems across all Android applications.

💾

Check Device Storage

Insufficient storage can cause unexpected behaviour. Ensure your device has adequate free storage space before using the application.

✅

Verify Permissions

Confirm that the application has been granted the permissions it requires. Go to device Settings → Apps → select the app → Permissions.

📋

Read the App FAQ

Each application's support page contains an FAQ covering the most common questions and troubleshooting steps specific to that app.

📸

Capture Screenshots

If a problem persists, take screenshots showing the issue before contacting us. Screenshots significantly speed up the support process.

Reporting Issues

How to Report a Bug Effectively

The more information you include, the faster we can identify and resolve the issue.

📱

Device Information

Include your device manufacturer, model name, and Android version (found in Settings → About phone).

📦

App Version

Include the version number of the Gliders Tech application (found in Settings or the About section within the app).

🔍

Steps to Reproduce

Describe exactly what you did before the issue occurred. Step-by-step instructions help us reproduce and confirm the problem quickly.
